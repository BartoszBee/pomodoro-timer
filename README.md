# Pomodoro Timer

Prosty, estetyczny i funkcjonalny **Pomodoro Timer** zbudowany w Next.js 16, React 19, TypeScript i Tailwind CSS 4.

## Funkcje

- Przełączanie między sesjami pracy i przerwy
- Pauzowanie i resetowanie timera
- Konfigurowalne czasy pracy i przerwy
- Powiadomienie dźwiękowe po zakończeniu sesji
- Pełna dostępność (ARIA) - obsługa czytników ekranu i nawigacji klawiaturą

## Technologie

- Next.js 16 (Turbopack)
- React 19
- TypeScript
- Tailwind CSS 4

## Uruchomienie

```bash
npm install
npm run dev
```

Aplikacja będzie dostępna pod adresem `http://localhost:3000`.
